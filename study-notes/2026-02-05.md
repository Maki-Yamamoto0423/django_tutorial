Django Test Client とは？
Django Test Client は、あなたの作成した Web アプリに対して「リクエストを送り、レスポンスを受け取る」という一連の流れを、ブラウザを使わずにテストするためのツール

なぜ「擬似」なのか？
通常のブラウザ（Chromeなど）は、ネットワーク（HTTP）を介してサーバーにアクセスしますが、Test Client は Django 内部のコードを直接呼び出します。

本物のブラウザ: ブラウザ -> ネットワーク -> Webサーバー -> Django

Django Test Client: Pythonコード -> Django

何ができるのか？
GET や POST リクエストの送信: フォーム送信やページ遷移のシミュレーション。

ログイン状態のシミュレート: client.login() で簡単にログイン済みユーザーとして振る舞えます。

レスポンスの検証: 戻ってきた HTML の内容、ステータスコード（200 OK や 404 Not Found）、使用されたテンプレートなどをチェックできます。

リダイレクトの追跡: ページ移動が正しく行われたかを確認できます。
